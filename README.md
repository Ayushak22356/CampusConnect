# Campus Event Management System

## Project Overview

The Campus Event Management System is a full-stack web application designed to streamline the organization and management of events within a college campus.

The system allows students to view and register for events while organizers can create and manage events. An admin panel ensures moderation by approving or rejecting event requests.

This project demonstrates full-stack development using modern technologies including React, Node.js, Express, MySQL and Sequelize ORM.

---

## Objectives

• Provide a centralized platform for campus event management  
• Allow students to easily discover and register for events  
• Enable organizers to create and manage events  
• Provide admin control for event approval and moderation
